# credential-security-hash-analysis
Hands-on cybersecurity lab repository focused on password security, hash analysis, and credential auditing. Includes practical exercises using tools like Hashcat and John the Ripper to understand hash types, cracking techniques, and password security concepts in controlled environments like TryHackMe.


---

## 📌 About This Lab

**Crack the Hash** is a hands-on **password auditing and cryptanalysis lab** from TryHackMe focused on identifying and recovering plaintext passwords from various hash formats.

It builds foundational understanding of **how passwords are stored, protected, and attacked in controlled security environments**.

---

## 🧠 Core Skills Covered

This lab focuses on practical skills used in real-world cybersecurity roles such as **SOC Analyst, Pentester, and Security Engineer**:

- 🔎 **Hash Identification & Analysis**
- 🔐 Understanding common hashing algorithms:
  - MD5
  - SHA-1
  - SHA-256
  - SHA-512
  - bcrypt
- ⚔️ Password recovery using:
  - Dictionary Attacks
  - Brute-force Attacks
- 🧂 Working with **Salted vs Unsalted Hashes**
- 🛠️ Using industry-standard tools:
  - Hashcat
  - John the Ripper
- 🎯 Selecting correct attack strategies based on:
  - Hash type
  - Available intelligence
- 🗂️ Understanding password storage mechanisms & authentication artifacts
